Here is how to view the MongoDB code for an aggregation pipeline:

1.      Connect to the server on MongoDB Compass.

2.      Open the database and collection and click on the aggregations tab.

3.      After creating your pipeline by adding stages, click Export Pipeline to Language.

Instructions:

Use MongoDB Compass and open movie collection in the movieDB database to answer the questions below. Include the query used (make sure to include all stages) and the results of the query.

Question

Query Used

Answer

1.      How many movies made in 1933 have a runtime of 100?

[{$match: {

runtime: 100,

year: 1933

}}, {$group: {

_id: null,

count: {

$sum: 1 }

}}, {$project: {

_id:0

}}]

 

count: 2

 

2.      Return the runtime and the year of the movie titled, Greed.

[{$match: {

title: 'Greed'

}}, {$project: {

_id:0,

runtime: 1,

year: 1

}}]

 

runtime: 140

year: 1924

 

3.      What movies have the longest runtimes? Return runtime, title, and year of the first 2 results only.

[{$sort: {

runtime: -1

}}, {$limit: 2}, {$project: {

_id:0,

runtime: 1,

title: 1,

year: 1

}}]

 

runtime: 399

title: "Les vampires"

year: 1915

 

runtime: 281

title: "Les Misrables"

year: 1934

 

4.      Return the average runtime of all movies by their year. Only return the year and average runtime of the shortest average runtime. (round the query result to one decimal if necessary)

[{$match: {

year: {

$gt: 1940

}

}}, {$group: {

_id: '$year',

avgruntime: {

$avg: '$runtime'

}

}}]

 

_id: 1942

avgruntime: 91.8

 

_id: 1941

avgruntime: 100.8

 

_id: 1943

avgruntime: 109

 

 

 

5.      Out of all the movies that were made in the year 1935, what is the maximum runtime?

(Hint: use $group stage)

[{$match: {

year: 1935

}}, {$group: {

_id: '$year',

maxruntime: {

$max: '$runtime'

}

}}, {$project: {

_id:0

}}]

 

maxruntime: 133

 

6.      Return the runtime, title, and year of the movie which is made in 1936 and with a runtime less than 90 minutes. Only return the first movie based on reverse alphabetical order by title.

 

[{$match: {

year:1936,

runtime:{$lt:90}

}}, {$sort: {

title: -1

}}, {$project: {

_id:0,runtime:1, title:1, year:1

}}, {$limit: 1}]

 

runtime: 84

title: "Three Smart Girls"

year: 1936

 

 

7.      What is the equivalent SQL code for the query you found in question 2?

SELECT runtime, year

FROM moviedb.movie

WHERE title= Greed;

 

8.      What is the equivalent SQL code for the query you found in question 6?

SELECT runtime, title, year

FROM moviedb.movie

WHERE year = 1936 AND runtime < 90

ORDER BY title DESC

LIMIT 1;
